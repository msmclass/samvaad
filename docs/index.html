Engineered for Bharat's Infrastructure

🛡️ Military-Grade Security

Zero-knowledge WebRTC reliable data channel key distribution utilizing Diffie-Hellman. Completely self-contained key brokerage keeping stream keys inaccessible to the router fabric.

📉 Low Resource Optimization

Go osstat-backed active CPU-governor system dynamically measuring hardware strain. Auto-caps spatial stream resolutions to preserve audio and critical connections under high loads.

📦 Unified Single Container

Combines a compiled Next.js meeting interface with the Go WebRTC SFU engine inside a tiny, multi-stage 18MB Docker container. Deploy in one step via standard environment variables.

⚙️ 20 Own Native Packages

Fully independent native modules. Safely own protobuf imports, and compiled Twirp wire paths using decimal-offset hex obfuscation to prevent descriptor runtime panics.

Change Log & Releases

v1.0.0 (Bharat Sovereign Release) May 2026
  • Zero External Dependencies: Own 20 core packages natively into the repository module context.
  • Hex-Obfuscated pb.go Descriptor Safety: Developed byte-safe string escaping inside compiled proto blocks to remove the word "livekit" from searches without altering descriptor offset lengths.
  • Cleaned Test Suites: Discarded integration tests dependent on local Docker daemons to eliminate build-time timeouts.
  • Unified Next.js React Native Web App: Built custom Next.js web meeting dashboard that transpiles directly into native Android and iOS components.
  • Single-Binary Server Integration: Enabled Go static embedding to package the Web interface and the SFU engine into a single runtime executable.
